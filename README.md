# Twitch Chat Bot

## Links
- [TMI.js Docs](https://github.com/tmijs/docs/tree/gh-pages/_posts)
- [Get bot token](http://twitchapps.com/tmi/)

## Installation
1. clone repository
2. `cd twitch-chat-bot`
3. `npm install`
4. create `config/default.json`
```json
{
  "channel": "<twitch channel>",
  "username": "<bot username>",
  "token": "<bot token>"
}
```

## Usage
1. `npm start`
